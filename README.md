# Trust Downloader

A simple and fast desktop application to download YouTube videos and audio.

## Download
You don't need to install Python to use this app.
1. Go to the **Releases** section on the right.
2. Download the `.exe` file.
3. Run it directly.

## Features
- **Video & Audio:** Download videos in MP4 or extract audio in MP3.
- **Quality Selection:** Choose your preferred resolution (from 480p to 1440p).
- **Easy to Use:** Just paste the link, select the format, and click download.
- **Save Anywhere:** Choose any folder on your computer to save your files.
